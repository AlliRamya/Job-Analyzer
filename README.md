# 📊 Job Market Analyzer

A Streamlit web app that analyzes job market trends using real-time data.

## 🚀 Features
- Fetches job data from API
- Displays job listings
- Filters by role
- Shows top skills
- Visualizes job trends

## 🛠 Tech Stack
- Python
- Streamlit
- Pandas
- Matplotlib

## ▶️ Run
streamlit run jobanalyzer.py
